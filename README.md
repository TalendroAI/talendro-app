# Talendro™ FULLSTACK_JS Starter (Create React App + React Router + Tailwind + Express)
Date: 2025-09-06

## Quickstart
1) `npm run install:all`
2) `npm run dev`           # starts Express backend (port 5000) + CRA dev server (port 3000)
3) Open http://localhost:3000

## Production build
- `npm run build` (builds React into client/build)
- `npm start` (Express serves API and static client build)

## Folders
- /client — Create React App (React Router + Tailwind)
- /server — Express API and static file server (serves /client/build in production)
